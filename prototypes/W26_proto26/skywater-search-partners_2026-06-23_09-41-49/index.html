Our read

That CFO-services search says your finance desk is handling more complex client work. It also sits beside a live job catalog with many practice areas. When lines grow, recruiters need faster intake, matching, and clear handoffs. SkyWater already uses Short List Search and candidate assessments to show fit. Tim's IT desk sees this pressure first when tools must support more recruiters. The risk is simple: strong searches slow down when data stays split.

Where we'd start

What we noticed

The mobile job-search page has slight side scroll.

Check the Matador form, table, and pagination styles first. This is a fast fix before bigger funnel work starts.

Useful next step

Remote roles are buried inside a long location list.

Add compact filters for Remote, Practice Area, and Salary where present. Keep full location search as backup.
