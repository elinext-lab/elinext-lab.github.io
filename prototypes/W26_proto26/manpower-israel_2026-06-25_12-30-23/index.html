Closing the SAP integration gap for Manpower Israel

Saw the Global Head of SAP signal. That usually means a new enterprise roadmap is coming, and local job, time, payroll and invoice systems must be ready.

Our Read

The SAP hire points to a serious back-office program. The hard part is likely not strategy only. It is connecting SAP with local systems that already run jobs, time reports, payroll and invoices. Manpower.co.il already has job search, CV upload, Smart Agent and login flows. Reviews for the ManpowerGroup Israel app mention crashes, login loops and false no-internet messages. If these flows are not mapped early, the SAP plan can slow local delivery.

Where We'd Start

What we noticed

The search page carries many candidate flows: filters, login, CV upload, Smart Agent, salary tools and source tracking.

Useful next step

Map which fields SAP should own, and which should stay in recruitment systems.

What we noticed

App reviews point to auth, connection state, job assignment and time-entry pain.

Useful next step

Add telemetry around login, network state, job loading and report submission.
