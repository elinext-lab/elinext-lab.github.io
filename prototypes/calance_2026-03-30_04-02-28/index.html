Design Logic

The wedge is small on purpose

This does not replace Genesys or rebuild Teams. It strengthens the layer Calance already owns: the moment where a loose support chat becomes a structured service event.

1. Capture once

Auto-fill asset identity, location, and prior incident history while the user is already in Teams. The first payload should be the best payload.

2. Route with context

Push severity, vendor dependency, and evidence into Genesys on creation so queue logic can do more than react to free text.

3. Expose ownership

Show SLA state, current owner, and vendor lane in one timeline so supervisors are not chasing status across multiple systems.
