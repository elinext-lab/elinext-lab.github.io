Our read

The signal says Prophesee is now selling a full system, not only a sensor stack. Mantara and Hearth make the story stronger, but they also raise delivery load. Hearth succeeds OpenEB and the Metavision SDK, so customers need a clean migration path. Open GitHub issues still show questions around GenX320, Raspberry Pi, Python packaging, and DAT-to-HDF5 flows. If that path stays unclear, pilots can slow down and support load can rise.
