Our read

The bottleneck sits behind the parking tap

The Parking Team hire says Pango is scaling a core payment domain. This is not only a staffing gap. It likely means more work on session state, location logic, billing checks, and support tools. Recent reviews name Pango activate parking, Pango Smart, and station exit flows. Users mention zone confusion, paid-service prompts, and hard account changes. If those paths stay fragile, engineers spend time on support instead of new mobility services.

Where we'd start

Two checks before adding more features

Zone confidence

Reviewers say GPS can know the place but still asks for manual zone choice. Log GPS, zone, and user override gaps before payment starts.

Paid-service guardrails

Users say paid prompts can appear while starting parking. Move offers away from the final tap, then audit cancellation state.
