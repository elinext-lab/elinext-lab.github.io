Our read

The Transformation Director signal points to a change program, not a simple hire. Connect Search already sells clarity, role calibration, and end-to-end ownership, so the digital path should prove the same care. The Search Jobs page is the main public product for candidates, but its no-script state says listings need JavaScript before anyone can view them. That can hide open roles from some crawlers, assistive setups, and locked-down browsers. If the job board feels lighter than the advisory promise, strong candidates may leave early.

Where we'd start

What we noticed

The job list has a weak fallback

Server-render the first page of roles, then let filters enhance it. This protects search, access, and trust if scripts fail.

What we noticed

The alert path asks for an account first

Test a save-search email step before registration. Passive candidates can show intent before they commit to an account.
